## 🏦📊 Bank Loan Analysis using SQL
### 📌 Project Title

Bank Loan Analysis using SQL

### 🧾 One-Line Summary

📈 An end-to-end SQL project to analyze bank loan applications, repayments, funded amounts, and loan risk performance.

### 📖 Overview

This project focuses on analyzing financial loan data using SQL to generate insights on loan applications, repayments, borrower profiles, and loan quality.

### ❓ Business Questions Addressed

- 1️⃣ Total loan applications
- 2️⃣ Funded amount and amount received
- 3️⃣ Average interest rate and DTI ratio
- 4️⃣ Good loans vs bad loans analysis
- 5️⃣ Loan status performance
- 6️⃣ Monthly loan trends
- 7️⃣ State-wise loan applications
- 8️⃣ Loan term analysis
- 9️⃣ Employment length analysis
- 🔟 Purpose and home ownership breakdown

### 🛠️ Tools & Technologies

- 💻 SQL
- 🗄️ MySQL
- 🧪 MySQL Workbench
- 🔗 Git & GitHub

### 📚 SQL Concepts Used

• Aggregate Functions
• CASE Statements
• GROUP BY
• ORDER BY
• Date Functions
• ALTER TABLE
• UPDATE
• Primary Key
• Data Cleaning

### 📊 Key Insights

- ✅ Identified total funded vs received amounts
- ✅ Compared good and bad loan percentages
- ✅ Found high loan demand states
- ✅ Tracked monthly application trends
- ✅ Analyzed borrower employment and home ownership patterns
